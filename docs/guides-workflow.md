# מדריכי תוכן - Workflow קצר

## הוספת מדריך חדש
1. משכפלים `content/guides/TEMPLATE.md`.
2. משנים שם קובץ ל-slug באנגלית (למשל `hermon-weekend.md`).
3. ממלאים Frontmatter + תוכן לפי הסעיפים.
4. בודקים מקומית בנתיב `/guides/<slug>`.

## דגשים תוכניים
- לשמור על עברית ברורה וקצרה.
- לא לכתוב המלצות לא חוקיות ללינה/ריקון מים אפורים.
- תמיד לכלול חלופת גיבוי.
- להזכיר שהסיוע בתכנון כלול חינם בהזמנה.
