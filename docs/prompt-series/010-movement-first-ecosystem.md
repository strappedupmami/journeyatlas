# Prompt/Response 010 - Movement-First Ecosystem

## Prompt summary
Need a travel/housing ecosystem optimized for movement and spontaneity.

## Response highlights
- Treat as ecosystem, not a single vehicle.
- Principles:
  - Always-ready
  - Redundancy
  - Modularity
  - Distributed bases
- Minimum viable stack:
  1. Sleep layer
  2. Power layer
  3. Internet layer
  4. Water/hygiene layer
  5. Work layer
  6. Storage layer
  7. Service layer
  8. Rules layer
