# Prompt/Response 003 - Cost Recovery Pricing

## Prompt
What should monthly subscription costs be in different phases depending on vehicle+build cost?

## Response highlights
- Use formula: `(unit_cost / recovery_months) + monthly_ops + margin`.
- Example anchor:
  - Unit cost: ₪120k
  - Recovery: 24 months
  - Ops: ₪1,650/month
  - Break-even: ~₪6,650 (rounded ~₪6,900)
- Phase pricing:
  - Phase 1: ₪7,500–₪8,000
  - Phase 2: ~₪6,500
  - Phase 3+: tiered pricing with cross-subsidy
