# Prompt/Response 005 - Trials Before Subscription

## Prompt
Starting directly with subscriptions may not work; start with short-term stays so people can test the concept.

## Response highlights
- Phase 1 = paid validation rentals.
- Trial pricing:
  - 3 days: ₪1,500 (or range around this)
  - 7 days: ₪3,000
  - 14 days: ₪5,000
- Goal of phase: data, not profit.
- Required loop:
  - Trial
  - Feedback survey
  - Identify discomfort/missing pieces
  - Offer trial spend as credit to first subscription month
- Finalize subscription after ~20–30 trials.
