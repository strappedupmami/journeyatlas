Prompt 002

Infrastructure-as-a-Service עם גישה קצרה וארוכה

יחידת החיים הניידת היא אותה תשתית, עם שני מודלי גישה: שהייה קצרה לחוויה/בדיקה ומנוי חודשי לשימוש מתמשך.

עמדת מיצוב

״אתם לא מוכרים ואן. אתם מוכרים תפקוד אנושי רציף.״

  • Always connected
  • Always powered
  • Always able to shower, cook, sleep
  • Never stranded when something breaks

עקרונות תכנון להורדת עלות

  • דגם בסיס אחיד + תכנית פנים אחידה
  • מבנה מודולרי מהיר בנייה ומהיר תיקון
  • סולאר + סוללה גדולה + shore power
  • מערכת מים/מקלחת/מטבח קבועה
  • משקל יעד < 3.5 טון להפחתת חיכוך רישוי

מודל גישה כפול

  • Short Stay: 3–14 ימים = מסע חווייתי
  • Long Stay: מנוי חודשי = מנוי מסע
  • הלקוח יכול להתחיל קצר ולעבור למנוי
  • חבילה כוללת: תחזוקה, ביטוח, חילוץ, ימי גיבוי לינה
  • תמחור מדורג לפי הכנסה (means-tested)
  • שהייה 3–12 חודשים להקטנת churn במסלול מנוי

מי משלם בפועל

  • יחידים (מנוי חודשי נגיש)
  • עמותות/רשויות/תורמים לאוכלוסיות החלשות
  • מעסיקים (דיור עובדים)
  • משפחות במעבר/משבר

מה שומר על רווחיות

  • Blueprint יחיד -> רכש בכמויות -> עלות יחידה יורדת
  • פרוטוקול תחזוקה פנימי עם מלאי חלקים סטנדרטי
  • Refurbish + Redeploy לכל יחידה כמה מחזורי שימוש
  • ניטור תקלות קריטיות להפחתת downtime
