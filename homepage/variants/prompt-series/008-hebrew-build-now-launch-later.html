Prompt 008

אתר עברי: בונים עכשיו, משיקים כשצריך

ההבטחה המרכזית: ״תכנון מסלול ועזרה בטיול כלול בחינם בכל הזמנה״, עם מנגנון תוכן סקיילבילי.

הצעת הערך

  • חופשה בלי מלונות (חופש תנועה אמיתי)
  • שקט בראש: מים/אפור/לינה/מקלחות מסודרים מראש
  • עזרה חינם למזמינים (לא Add-on בתשלום)
  • מדיניות איכות ברורה: ללא עישון, חוקיות מלאה

סגמנטים מרכזיים

  • זוגות/חברים לבריחה קצרה
  • צעירים אחרי צבא
  • אנשים שצריכים שבוע ריסט מהעבודה
  • מטיילים שרוצים עצמאות עם תמיכה

מבנה אתר Phase 1

  • דף הבית: הבטחה + חבילות + CTA
  • חבילות ומחירים: מסע חווייתי + מנוי מסע
  • מדריכים בישראל: מנוע SEO ותמיכה
  • FAQ: הריגת התנגדויות פומבית
  • מדיניות: ללא עישון, ניקיון, מים/אפור
  • צור קשר: WhatsApp קודם

Travel Help חינם בלי להישחק

מודל דו-שכבתי:

  • Public Guides = 80% מהערך לכלל הגולשים
  • Customer Pack = מפות/צ׳קליסטים אחרי הזמנה

Build Now, Launch Later

  • Staging/Preview בלבד כרגע
  • Robots noindex בסביבת בנייה
  • Repo נקי עם commit קטן ותדיר
  • מעבר לפרודקשן רק כשיש בשלות

יום עבודה מומלץ (Minimal Yasha Load)

  1. הרצה מקומית
    Yasha מריץ פקודות; GPT כותב את הקוד והקבצים.
  2. יישור עברית + RTL
    זכייה ויזואלית מיידית שמייצרת מומנטום.
  3. 3 מדריכים ראשונים
    חופים / צפון / מדבר עם דגש מים/אפור חוקי.
  4. Push + Preview
    Commit, GitHub, ו-Preview URL לבדיקות בלבד.
